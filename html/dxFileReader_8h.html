Detailed Description

Reads an OpenDX data file.

This library implements a light weight reader for the native OpenDX data format. The complete standard is not necessarily implemented.

Author:
David J. Warne (david.warne@qut.edu.au)
High Performance Computing and Research Support
Queensland University of Technology
